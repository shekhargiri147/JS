Including the JavaScript

We can include JavaScript in our HTML in two ways:
1. Writing the code in our HTML
2. Including it as a link to an external file
For the most part, we will include the JavaScript as an external file.

The Script Tag
The "script" tag is what we use to includes our JavaScript. It's a lot like the "link" tag we've already been using to include our CSS files.
